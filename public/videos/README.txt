# Video Files for Header

Place the following video files in this directory:

1. `header-video.mp4` - Main video for the header background (16:9 aspect ratio, 1920x1080 recommended)
2. `header-poster.jpg` - Poster image to display while the video is loading

## Recommended Video Specifications

- Format: MP4 with H.264 encoding
- Resolution: 1920x1080 (Full HD) or higher
- Duration: 15-30 seconds
- Aspect Ratio: 16:9
- File Size: Optimize for web (2-5MB recommended)

## Video Content Suggestions

The video should showcase:
- Modern paint applications
- Insulation installation processes
- Beautiful finished spaces
- Professional work environments

Keep the video subtle enough to not distract from the text overlay.
